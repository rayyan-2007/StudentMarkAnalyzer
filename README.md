# 🎯 Student Mark Analyzer (JavaScript)

## 📌 About

The **Student Mark Analyzer** is a basic JavaScript project designed to analyze student marks using core programming concepts like loops, arrays, and functions.

This project focuses on building strong logic by solving real-world problems such as finding the highest mark, calculating averages, and validating input data.

---

## 🚀 Features

* Find the **highest mark**
* Find the **lowest mark**
* Calculate **average marks**
* Count **passed students**
* Validate marks (0–100 only)
* Skip invalid inputs using logic control

---

## 🛠️ Technologies Used

* JavaScript (Core concepts)
* Arrays
* Loops (`for`)
* Functions

---

## 📂 Example Input

```js
let marks = [45, 67, 89, 32, 76];
```

---

## 🧠 Concepts Covered

* Looping through arrays
* Conditional statements (`if`)
* Function design
* Input validation
* Problem-solving logic

---

## ⚠️ Edge Cases Handled

* Empty array
* Invalid marks (less than 0 or greater than 100)
* All students failing
* Duplicate values

---

## 📈 Learning Outcome

This project helps in:

* Improving logical thinking
* Understanding how loops work in real scenarios
* Writing clean and reusable functions
* Avoiding common beginner mistakes

---

## ▶️ How to Run

1. Copy the code into a `.js` file
2. Run using:

```bash
node filename.js
```
